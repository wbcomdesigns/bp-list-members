This Plugin will extend the BuddyPress functionality and will allow to create friend groups/lists.

== Description ==

This plugin will add an extended feature to the big name “BuddyPress” that will allow to create friend groups/lists and list all selected members to the specific group/list.

== Installation ==
This section describes how to install the plugin and get it working.

1. Download the zip file and extract it.
2. Upload `bp-list-members` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the \'Plugins\' menu.
4. Alternatively you can use WordPress Plugin installer from Dashboard->Plugins->Add New to add this plugin
5. Enjoy
